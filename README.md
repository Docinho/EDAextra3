# EDAextra3

InsereOrdenadoPrimeiro: Inserção ordenada do primeiro elemento de uma lista já ordenada

InsereOrdenadoUltimo:  Inserção ordenada do último elemento de uma lista já ordenada

MergeSort: ordena um array de inteiros utilizando o merge sort e imprimindo o status de cada etapa do algoritmo

ParticionaComPivot: particiona uma sequencia utilizando o primeiro elemento como pivot. No final, todos os elementos à esquerda são menores que o pivot, e todos à direita são maiores. A sequencia manterá a ordem original, não ficando, necessariamente, ordenada no final. Os elementos iguais ao pivot ficarão juntos a ele.

SelectionRecursivo: encontra o menor elemento do array e coloca-o na posição correta(Selection Sort). Utiliza recursão. Imprime o status a cada "modificação" feita no array
